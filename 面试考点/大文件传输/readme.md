# 大文件上传
1. 用户选中大文件
2. 前端将大文件切分成多个小文件 （文件切片）: 借助file.slice方法，将blob文件切片，将多个切片文件转为from表单数据（16进制），然后分批上传
3. 前端将多个小文件并行上传到后端 : 
4. 后端接受多个小文件并合并成大文件： 后端接受到多个片段，将这些片段存在后端的一个临时目录中
5. 后端保存合并后的大文件：当前端上传都成了，前端向后端发请求，告知后端片段传输完毕，后端执行文件合并任务


# file.slice()
1. 语法：file.slice (start, end)
2. 作用：返回一个新的 File 对象，包含从 start 到 end（不包括 end）的文件数据。
3. 参数：
    - start：可选，开始提取文件数据的字节位置，默认值为 0。
    - end：可选，提取文件数据结束的字节位置，默认值为文件的大小。
4. 返回值：一个新的 File 对象，包含从 start 到 end（不包括 end）的文件数据。
