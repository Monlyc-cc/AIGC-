# 懒加载

1. 如果一次性要加载许多项目，但是视窗根本展示不下， 则可以不加载视窗外的组件，等到组件出现在视窗内时，则可以加载出来，称之为懒加载



# import 的多种用法
1. 关键字用法，写在文件顶部
2. 函数式用法，/* import('./Demo') */
    - 配合lazy 关键字,可以实现组件的懒加载

# react-lazyload
1. 当内容出现在可视区域内,才加载资源(仅对于图片资源)
2. 用 react.lazy() 包裹的组件,组件文件就不会被提前读取,而是当组件被使用时在会读取.
    配合react-lazyload时,可以实现组件到达可视区域时,才被读取并加载
3. 属性的使用:
    - offset(num) 预加载,当前lazyload模块距离可视窗口 num px时,则提前加载
