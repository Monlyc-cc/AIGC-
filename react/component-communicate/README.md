# 组件通讯
1. 父子组件: 将父组件的变量，传递给子组件使用
- 方式： 
    - 父组件直接在子组件标签上绑定属性值 
    - 子组件通过参数props 接收,props 接收到的数据是只读的。
2. 子父组件：将子组件传递给父组件使用
    - 父组件提前向子组件传递一个函数，以这个函数创造一个子组件向父组件传递数据的通道
    - 当子组件调用这个函数时，就可以利用该函数操作父组件内部的数据。
        - 如果需要更新父组件状态，在页面中重新渲染父组件，可以使用钩子函数（useState(obj)）
        - let [obj,setobj]=useState(obj);
        - 这样就可以在父组件中定义一个变量obj与用来修改obj的函数setobj
        - 调用setobj 时，既可以修改obj变量，有可以使得父组件重新渲染一次。
3. 兄弟组件传值
-  子组件1给父组件传值
-  父组件再给子组件2传值
 （仓库）

4. 跨组件通信(父向后代传值)
1. 父组件用creatContext创建一个上下文对象，并用provide传递一份数据
2. 需要接收数据的那个子组件用useContext 接收这份数据
