# 构造函数
1. 当我们需要批量构建对象时，可以使用构造函数。
2. 当一个函数被new调用时，我们称之为构造函数。
3. js中构造函数与普通函数没有本质区别，

# new干了什么？new的原理
1. 在函数中创建了一个 this 对象
2. 执行构造函数中的内容，（给this对象添加属性值）
3. 返回this 对象
4. 如果构造函数内return 一个对象，则覆盖默认对象。

# 包装类
1. 当用户定义一个字面量时，v8默认执行的还是：new Xxx（），v8看来不存在原始类型，原始类型都被包装为引用类型
2. 因为js是弱类型语言，只有在赋值语句执行时，才会判断数值的类型，当该值被判定为原始类型，就会自动将包装对象上添加的属性移除。
3. 一个包装类的实例对象，在参与运算时，会自动拆箱成原始类型
