# let const var 
1.  var 可以重复声明变量，let const 不行
2.  var 在全局声明的变量默认会添加在 window 上， let const 不会
3. var 声明变量存在“声明提升”，let const 不存在。
4. let const 声明的变量配合{} 形成块级作用域
