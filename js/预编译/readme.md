2. js 代码被编译时，声明部分会提升到当前作用域顶部

# 函数预编译

1. 创建函数的执行上下文 AO对象
2. （将声明的变量与形参塞入Ao对象，属性名为变量名，属性值为undefined）找将声明的变量塞入Ao对象，属性名为变量名，属性值为undefined）形参和变量声明，将形参和变量声明作为Ao的属性名，值为undefined
3. （将形参值塞入Ao对象，形参名为属性名，形参值为属性值）将实参与形参统一
4. （将声明的函数塞入Ao对象，属性名为函数名，值为函数体。）在函数体内找函数声明，函数名作为Ao的属性名，值为函数体。

# 全局预编译
1. 创建全局执行上下文 Go对象
2. 找变量声明，将变量塞入Go，属性名为变量名，属性值为undefined
3. 找函数体声明，将函数塞入Go，属性名为函数名，值为函数体
