# 数据类型
- 原始类型（简单类型）//通常占据空间很少
1. string（字符串）
2. number（数字）
3. boolean（布尔）
4. undefined
5. null
es6版本后新增
6. symbol 定义一个独一无二的值，Symbol类型的值无法用于计算
7. Bigint大整型,'数字后带n' ，例如‘12345667n’


- 引用类型（复杂类型）//可能会占据巨大的内存空间，但栈空间不能设计的很大，所以会将引用类型数据存储在堆空间中，然后栈空间内，对象指向堆中地址
1. 数组,线性存储结构
2. 对象（字典）,非线性的存储结构
3. 函数,在js中也是一种数据类型,可以被当成对象来调用。

1. 原始类型的值存储在，调用栈中。
2. 引用类型的值，因为可能非常大，所以将数据存储在堆中，再将堆中的引用地址存在栈中。
