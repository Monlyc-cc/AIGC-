# 文件流
- html中的标签分为三类：
1. 块级元素（div）：默认占据一整行，或是占据父容器宽度的100%。
2. 行内块级元素（img、input）:可以同行展示，且被设置宽高
3. 行内元素（span、a）:无法被设置宽高

- 浏然器在加载html容器时，默认从上往下，从左往右排列这些元素。

# 定位布局

#修改文件流
1. 相对定位:position:relative;以容器自身为参考点，进行定位。



# 脱离文件流
2. 绝对定位:position:absolute;以拥有定位属性的父容器为参考标准，进行定位，如果没有则层层网上找拥有定位属性的父容器，直到body标签为止，并以body为参考标准.父容器在子容器使用绝对定位时，子容器的定位参考点为父容器。
3. 固定定位：fixed;以屏幕可视区域为参考点，以屏幕可视区域为参考标准。（始终在页面的某个位置）
4. 粘性定位：position:sticky:以屏幕为参考标准，当页面滚动到阈值时，背定位元素就像固定在该位置上。（当页面划走时，跟随向下）,但不会脱离父容器。
