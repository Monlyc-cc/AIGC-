# 大模型是怎么理解人类语言？( LLM )
 - 大模型会把每一个词用数字来表示

 # 通俗来讲
 
 假设 
- 从地位角度
  国王   2
  女王   2

- 从性别角度
  国王   1
  女王   2

        性别   地位
  国王   1      2
  女王   2      2
  男人   1      1
  女人   2      1

  可知 国王-男人+女人=女王
  即（向量运算）
  由此利用数字运算替代逻辑运算

  gpt-1 768个数字表示一个中文词 
  gpt-2 1600个数字表示一个中文词
  gpt-3 12280个数字表示一个中文词
  
  
  deepseek v3 7168数字 （性能与gpt3相似）：用少量的词表达更准确的意思，则性能更高
  
  本质：(词嵌入) 
   - 创建一个n维空间表达一个词
